Nombre del proyecto: Sistema de manual tecnico
Diseñador:Diana Carolina Serrano Tovar

Objeto del proyecto:

Maqueta en HTML Y CSS para la implemnetacion de un manual tecnico en cualquier area requerida.

Modificaciones:

Cambio de titulo o subtitulos: Modificar el contenido de las etiquetas <h1><h2>

Agregar imagenes propias del proyecto: Guardar estas en el fichero IMG,  e incorporarlas en el html (Renplazando las que ya estan en la etiqueta <img>). manejar un tamaño entre los 300px a 450px.

Agregar mas temas: Dentro de la etiqueta <nav> se encuentra el listado de temas, si desea agregar otro debe agregar una etiqueta <li> y el enlace con la etiqueta <a>, se debe crear un archivo HTML al que sera redirigido.

Agregar mas subtemas: Dentro de la etiqueta <div class> puede  si desea agregar otro debe agregar una etiqueta <li> y el enlace del parrafo que desea conectar.
